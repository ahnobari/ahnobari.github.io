```
│   └── style.css       # Styles
├── js/

    └── PDFs/

```

## Deployment

This site is deployed via GitHub Pages. Push to `main` branch to deploy.

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Then visit `http://localhost:8000`
